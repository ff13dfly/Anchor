# Anchor教程

- 这个教程将一步步的说明怎么使用Anchor来进行Web3.0的开发，跟着教程走，你只需要1天的学习时间，就可以开始使用Anchor来进行开发了。

- Anchor Network提供了测试网络来模拟真实的运行环境，你可以通过以下几种方式来访问

    1. Websocket服务器：`wss://dev.metanchor.net`
    2. Substrate节点：`[link](https://polkadot.js.org/app)`
